# SocialSense

**SocialSense** is a dynamic social platform that allows users to share, comment on, and react to memes in real time. Equipped with Socket.IO for live notifications and AWS-hosted machine learning models, SocialSense offers a vibrant community for meme enthusiasts to explore content and engage with each other respectfully and enjoyably.

## Features

- **Real-Time Notifications**: Instant updates via Socket.IO for likes and comments to keep users engaged.
- **Content Moderation**: ML models categorize comments into neutral, offensive, or hate speech, highlighted by blue, yellow, and red colors, ensuring respectful interactions.
- **Search Functionality**: Users can find other users and memes with an advanced search feature.
- **Profile Customization**: Users can personalize their profiles and share with the community.
- **Activity Feed**: A feed that chronologically lists users' past posts and replies.
- **OCR Text Extraction**: ML-powered OCR to extract text from images for content appropriateness.
- **Notification Counters**: Badge icons accumulate notifications to alert users of new activity.
- **AWS-Hosted Infrastructure**: Backend architecture hosted on AWS ensures scalability, reliability, and security.
- **Adaptive Themes**: Automatically adjusts the app theme to match the user's device settings, enhancing visual comfort and personalization.

## Technologies Used

- **Frontend**: Flutter, Dart
- **Backend**: Node.js, Express.js
- **Real-Time Communication**: Socket.IO
- **Machine Learning**: TensorFlow, PyTorch
- **Database**: MongoDB,Firebase
- **Hosting/Infrastructure**: AWS EC2, AWS S3, Render
- **Other Tools**: Docker

## Installation


Install and run this project by following these steps:

- Clone this repository:

```bash
git clone https://github.com/Mayank-kr-sh/SocialSense.git
```

- Navigate to the project directory:

```bash
cd Application/social_sense
```

- Install dependencies:

```bash
flutter pub get
```

- Run the app:

```bash
flutter run
```
